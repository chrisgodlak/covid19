# covid19
covid19 things

Stata graphics files for GHS index (https://www.ghsindex.org/) vs various covid-19 metrics (https://github.com/owid/covid-19-data)

_ee: East Europe_

_we: West Europe_

_eoy: end of year_

_avg: average_

_med: median_

Software: Stata 16.1

Analysis on https://godlak.substack.com/
